# 📂 Source Files

This folder contains the source code and dataset used to perform the complete Diwali Sales Analysis project.

## 📁 Contents

| File | Description |
|------|-------------|
| 🐍 **Diwali Sales Analysis.ipynb** | Jupyter Notebook containing data cleaning, preprocessing, exploratory data analysis (EDA), visualizations, and business insights. |
| 📊 **Diwali Sales Data.csv** | Original dataset used for the analysis. |

## 📌 Workflow

1. Load the dataset.
2. Perform data cleaning and preprocessing.
3. Conduct Exploratory Data Analysis (EDA).
4. Create statistical visualizations.
5. Generate actionable business insights.

> **Note:** Open the notebook in **Jupyter Notebook** or **Google Colab** and ensure all required Python libraries are installed before running the project.
