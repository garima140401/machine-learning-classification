
# 🛍️ Diwali Sales Analysis using Python

## 📌 Project Overview
This project focuses on analyzing Diwali sales data to uncover valuable business insights through data cleaning, exploratory data analysis (EDA), and visualization. The analysis helps businesses understand customer purchasing behavior and identify key sales trends to support data-driven decision-making.

---

## 🎯 Key Objectives
- Clean and preprocess raw sales data.
- Perform exploratory data analysis (EDA) to discover hidden patterns.
- Analyze customer demographics and purchasing behavior.
- Identify top-performing products and categories.
- Generate actionable business insights for improving sales and inventory planning.

---

## 🚀 Project Learnings

### 🧹 Data Cleaning & Preprocessing
- Cleaned and transformed raw data by handling missing values, duplicate records, and inconsistent data formats.
- Prepared the dataset for accurate analysis and visualization.

### 📊 Exploratory Data Analysis (EDA)
- Conducted comprehensive EDA using:
  - **Pandas** for data manipulation
  - **NumPy** for numerical computations
  - **Matplotlib** for data visualization
  - **Seaborn** for advanced statistical plots
- Explored customer behavior, sales trends, and purchasing patterns through various visualizations.

### 👥 Customer Insights
- Identified potential customer segments based on:
  - Gender
  - Age Group
  - State
  - Occupation
- Derived insights to help businesses improve customer targeting and marketing strategies.

### 📦 Sales & Product Analysis
- Identified the best-selling products and product categories.
- Analyzed purchasing trends to support:
  - Inventory management
  - Demand forecasting
  - Sales optimization
  - Business decision-making
