# 🖼️ Project Assets

This folder contains all visual resources used throughout the project documentation. These assets help illustrate the data analysis process, highlight key findings, and provide a clear visual representation of the project's results.

## 📁 Contents

| Asset | Description |
|--------|-------------|
| 👥 Gender Analysis | Visualization of customer purchases by gender. |
| 🎂 Age Group Analysis | Sales distribution across different age groups. |
| 💍 Marital Status Analysis | Customer purchasing behavior based on marital status. |
| 🌍 State-wise Sales Analysis | Sales performance across different states. |
| 💼 Occupation Analysis | Purchasing trends across various occupations. |
| 🛍️ Product Category Analysis | Performance comparison of product categories. |
| 📦 Top Products Analysis | Best-selling products based on purchase volume. |

## 🎯 Purpose

These visualizations summarize the results of the Exploratory Data Analysis (EDA) and help communicate key business insights in a simple, intuitive, and visually engaging manner.

> **Note:** All images are generated from the analysis performed in the Jupyter Notebook and are referenced throughout the main project documentation.
